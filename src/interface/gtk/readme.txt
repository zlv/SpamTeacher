SpamTeacher v0.5 win32
Программа для рассылки сообщений с произвольного сервера smtp,
Создана с помощью gtk+ и minGW.
Создатель: Евгений Лежнин <z_lezhnin@mail2000.ru>
В ней используется sendEmail v1.56 (http://caspian.dotconf.net).
Инструкция:
1. Введите всю необходимую информацию в окна настроек("Настройки 
отправителя", "Настройки получателя", "Настройки вложений").
2. Введите заголовок письма и сообщение. Вы можете вставлять 
шаблоны %name и %ms|%fs в сообщение с помощью кнопок "Имя" и 
"Пол" соответственно, эти шаблоны будут заменены текстом для 
каждого получателя индивидуально: шаблон %name будет заменён 
на имя получателя, а шаблон пола будет заменён на одну из 
указанных вами строк в зависимости от пола.
3. Отправляйте сообщение! (Во время отправки
сообщений программа SpamTeacher будет показывать что пишет
программа sendEmail в стандартный поток).
